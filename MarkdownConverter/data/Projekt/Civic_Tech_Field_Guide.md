# Civic Tech Field Guide

## Kurzbeschreibung
Der Civic Tech Field Guide ist die weltweit größte Sammlung von Projekten die Technologie für das Gemeinwohl einsetzen. Die Plattform katalogisiert insbesondere den Bereich der Public Interest AI und stellt alle Daten offen unter einer Creative Commons-Lizenz zur Verfügung.
