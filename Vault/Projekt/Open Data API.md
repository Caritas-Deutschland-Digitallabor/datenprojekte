# Open Data API

## Kurzbeschreibung
Die Open Data API ist eine Programmierschnittstelle, die vom OK Lab Flensburg entwickelt wird, um den einfachen Zugriff auf öffentliche Daten von Gemeinden, Verwaltungen und anderen Quellen zu ermöglichen. Diese Daten können dann in Anwendungen, Websites und Visualisierungen integriert werden.
