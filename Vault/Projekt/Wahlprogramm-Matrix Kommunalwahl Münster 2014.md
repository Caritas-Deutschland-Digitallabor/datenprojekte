# Wahlprogramm-Matrix Kommunalwahl Münster 2014

## Kurzbeschreibung
Die Wahlprogramm-Matrix ist ein Tool zum automatisierten Vergleich von Abschnitten in Wahlprogrammen mittels NLP. Es hebt ähnliche Passagen farblich hervor und unterstützt so bei der Entscheidungsfindung zur Kommunalwahl Münster 2014.
