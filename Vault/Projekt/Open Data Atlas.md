# Open Data Atlas

## Kurzbeschreibung
Der Open Data Atlas bietet eine Übersicht über alle Open Data Portale und Angebote in Deutschland, Österreich und der Schweiz. Er ermöglicht eine komfortable Suche, zeigt viele Informationen und visualisiert die Daten auf einer Karte.
