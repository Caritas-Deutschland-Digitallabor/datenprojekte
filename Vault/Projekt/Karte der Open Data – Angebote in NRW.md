# Karte der Open Data – Angebote in NRW

## Kurzbeschreibung
Die Karte bietet eine Übersicht über alle aktuellen kommunalen Open Data-Angebote in Nordrhein-Westfalen. Sie zeigt, welche Kommunen ein Open Data-Portal besitzen, welches System oder Lizenz sie verwenden und ob eine Anbindung an das Open.NRW-Portal besteht.
