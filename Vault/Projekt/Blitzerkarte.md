# Blitzerkarte

## Kurzbeschreibung
Die Blitzerkarte visualisiert täglich Straßen, auf denen die Stadt möglicherweise mobile Geschwindigkeitskontrollen durchführt. Die Daten werden vom Ordnungsamt der Stadt Essen bereitgestellt.
