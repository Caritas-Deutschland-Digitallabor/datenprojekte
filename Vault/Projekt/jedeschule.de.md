# jedeschule.de

## Kurzbeschreibung
jedeschule.de sammelt alle Schuldaten Deutschlands auf einer Plattform. Das Projekt stellt eine zentrale API und wöchentliche CSV-Dumps der Schuldaten bereit, die durch individuelle Webscraper aktualisiert werden.
