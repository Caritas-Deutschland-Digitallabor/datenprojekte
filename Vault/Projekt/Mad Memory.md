# Mad Memory

## Kurzbeschreibung
Mad Memory ist ein Online-Memory-Spiel, das dazu einlädt, sich spielerisch mit Wörtern aus dem Rheinischen Mitmachwörterbuch und dem Sächsischen Wörterbuch zu beschäftigen.
