# Visualisierung der Polizeiticker Meldungen der Leipziger Volkszeitung

## Kurzbeschreibung
Dieses Projekt bietet eine Weboberfläche zur Visualisierung von Polizeiticker-Meldungen der Leipziger Volkszeitung. Es umfasst einen Crawler zur Datenerhebung, Named Entity Recognition für die Analyse und Elastic Search für die Speicherung und Suche, um die Daten auf einer interaktiven Karte darzustellen.
