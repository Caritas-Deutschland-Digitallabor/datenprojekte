# Ergebnisse der Kommunalwahl 2014

## Kurzbeschreibung
Dieses Projekt visualisierte die Ergebnisse der Kommunalwahl 2014 in Münster live. Es zeigte die Gewinner der Direktmandate je Wahlbezirk auf einer interaktiven Karte.
