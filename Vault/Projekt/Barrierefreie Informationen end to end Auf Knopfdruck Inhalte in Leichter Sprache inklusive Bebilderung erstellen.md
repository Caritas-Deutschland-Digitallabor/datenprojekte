# Barrierefreie Informationen "end to end": Auf Knopfdruck Inhalte in Leichter Sprache inklusive Bebilderung erstellen

## Kurzbeschreibung
Leicht verständlicher Text ist nur ein Element von barrierefreien Inhalten in Leichter Sprache. Mit dazu gehört ebenfalls eine anschauliche Bebilderung! Diese wollen wir mithilfe von KI automatisieren, sodass Informationen für alle gleichberechtigt zugänglich sind.
