# Meta API für Elektronische Fahrplanauskunft

## Kurzbeschreibung
Ein JSON-API, das das bestehende Application Programming Interface (API) der Elektronischen Fahrplanauskunft (EFA) in eine RESTful API umwandelt. Dies soll die Entwicklung von Anwendungen durch Vereinfachung des Zugriffs erleichtern.
