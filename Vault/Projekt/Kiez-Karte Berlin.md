# Kiez-Karte Berlin

## Kurzbeschreibung
Die Kiez-Karte Berlin visualisiert offene Daten aus Berlin auf einer interaktiven Karte. Sie soll Bürgerbeteiligungsmöglichkeiten einbinden und städtische Informationen bürgernah darstellen.
