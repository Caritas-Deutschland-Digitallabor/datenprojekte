# codo – Das Daten-Tool für Mentoringprojekte

## Kurzbeschreibung
codo ist ein Open Source Daten-Tool (Web-App) für Mentoringprojekte, das alle Daten an einem zentralen Ort bündelt. Es erleichtert die Koordination, Dokumentation und das Reporting für Mentoringprojekte.
