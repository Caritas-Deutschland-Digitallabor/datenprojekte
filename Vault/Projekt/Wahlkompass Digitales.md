# Wahlkompass Digitales

## Kurzbeschreibung
Der Wahlkompass Digitales schafft Transparenz im Wahlkampf, indem er Antworten deutscher Parteien auf digitalpolitische Fragen vergleicht. Das Tool ermöglicht den Vergleich von Wahlprogrammen zu Digitalthemen in verschiedenen Politikbereichen.
