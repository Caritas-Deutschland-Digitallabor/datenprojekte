## Obsidian-Vault to Cosma-Graph
1. `python3 obsidian2cosma.py -i CaritasDigitaleLandkarte-Vault -o CaritasDigitaleLandkarte-cosma/data`
2. go to the `data/CaritasDigitaleLandkarte-cosma/data` folder and type `cosma c` for new project or `cosma m` if you want to generate the html document from the converted vault
